   # DiveLikeAHuman

 ## 一、项目概述
 面向水下自主系统，旨在构建具有“类人决策特性”的导航与避障框架，用于复杂水下环境中的行为选择与路径引导。

 ## 二、核心功能

 ### 1. 环境表征与情景建模
 - 对周围目标与障碍物进行结构化描述
 - 区分静态障碍物与动态个体，并提取关键交互要素
 - 关注有限数量的高优先级目标以降低决策复杂度

 ### 2. 行为评估与动作选择
 - 对一组基础行动模式进行综合评估（安全性与效率权衡）
 - 输出动作偏好分布及推荐行为
 - 提供决策意图与解释信息，支持结果分析与可解释性研究

 ### 3. 控制引导与系统协同
 - 将高层决策结果转换为下层控制系统可用的约束与引导参数
 - 支持与现有仿真环境和控制器的集成
 - 适用于闭环仿真与对比实验设计

 ## 三、典型应用场景
 - 水下机器人与 AUV 自主导航与避障研究
 - 多智能体交互与协同行为建模
 - 决策算法原型验证与方法对比实验

 ## 四、主要特征
 - 面向水下多目标、多障碍场景的专门设计
 - 强调决策过程的可解释性与分析友好性
 - 模块化结构，便于扩展不同感知、决策与控制实现
